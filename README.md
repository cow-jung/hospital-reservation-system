## 📌 프로젝트 소개

병원 예약 관리 시스템은 환자와 관리자가 콘솔 환경에서 손쉽게 병원 예약을 관리할 수 있도록 만든 파이썬 프로그램입니다.
회원가입/로그인부터 예약 신청, 조회, 취소, 관리자 관리, 통계 리포트까지 병원 예약 업무의 전 과정을 다룹니다.

> 팀 프로젝트로 진행되었으며, 데이터는 CSV 파일을 기반으로 저장·관리됩니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 👤 회원가입 / 로그인 | 사용자 인증 및 계정 관리 |
| 📅 예약 관리 | 예약 신청, 조회, 수정, 취소 |
| 🛠️ 관리자 기능 | 예약 현황 관리, 회원 관리 |
| 📊 리포트 / 통계 | 예약 현황 통계 및 리포트 출력 |
| 💾 데이터 저장 | CSV 파일 기반 데이터 저장 및 불러오기 |
